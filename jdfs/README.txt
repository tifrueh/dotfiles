                                      jdfs

                                      ---

                  Example Root: /home/user/.zsh/plugins/jdfs/

This module essentially contains a 'zsh' plugin for integrating with 'jdfs',
providing the following shell functions for convenience:

* 'jcd': Change directory by ID.
* 'jls': List the contents of a directory selected by ID.
* 'jpushd': Push a directory selected by ID to the directory stack.
